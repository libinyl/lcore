--------------历史过程--------------

CPU上电,BIOS 自检
BIOS 从 #0 sector 把 bootloader 加载到 0x7c00 并执行

bootloader:
	A20地址线打开

	设置 C 语言环境, 栈相关指针
	bootmain 将 KERNEL 的 elf header 从第二块扇区加载到内存 64KB 处
